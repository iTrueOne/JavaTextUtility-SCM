# Java Text Utility Tool

This is a simple Java utility tool developed for the Software Configuration Management course.

## Tool Description

This tool supports text analysis features, including palindrome checking and other text utility operations.

## Features

- Count characters in a text.
- Count words in a text.
- Reverse input text.
- Check whether the input text is a palindrome.
